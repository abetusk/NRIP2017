Harvard Personal Genome Project

The Harvard Personal Genome Project provides an open data resource to the scientific community by sequencing human genomes and making them publicly available.

I'm a staff member. I've donated my genome publicly to science and so can you.

Lightning Genome Encoding

I'm currently working on encoding whole genomes into about 30Mb (about the size of an MP3 album) with a focus on whole population analysis and machine learning.
